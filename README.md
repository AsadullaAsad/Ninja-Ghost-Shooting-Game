# Ninja Ghost Shooter - 2D Shooting Game

A fun 2D arcade shooting game built with **C++**, **OpenGL**, and **GLUT**.

### Features
- Two playable characters: **Little Ninja** & **Little Ghost**
- Shoot falling enemies (eggs)
- Texture-based graphics with stb_image
- Multiple game states (Menu, Playing, Instruction, Game Over)
- Smooth 60 FPS gameplay

### Controls
- `A` / `D` → Move Left / Right
- `SPACE` → Shoot
- `N` → Play as Ninja
- `G` → Play as Ghost
- `I` → Instructions
- `R` → Restart

### Technologies
- C++ 
- OpenGL + GLUT
- Texture Mapping
- Collision Detection
- Game State Management
